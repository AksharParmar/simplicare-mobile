# SimpliCare Agent Guide

## Product
SimpliCare is a privacy-first medication management app.

## Target Users
- Busy professionals
- People managing multiple medications or chronic conditions

## MVP Screens
- Onboarding
- AddMedication (scan/manual)
- Today
- Copilot
- History/Export
- Settings

## Hard Rules
- Local-first storage (no account required for core features)
- Encrypt sensitive user data at rest
- No analytics by default
- AI responses must be source-grounded and must not diagnose or prescribe
- Do not add features outside MVP scope unless explicitly requested
